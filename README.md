# PHP-oneluck

このソフトはrubyで作成したone_luckをPHPで実装する為に勉強用として作成します。
# バージョン
PHP7

# 前提
- vagrant
- virtualbox

# Setup
```
vagrant up
vagrant ssh

sudo apt-get update
sudo apt-get install nginx php-fpm
sudo apt-get install mysql-server mysql-client
sudo apt-get install php-mysql php-mbstring
```
